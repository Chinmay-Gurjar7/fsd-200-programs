CSS Hamburger Menu

Click the hamburger icon (top right on mobile, or resize window) to open the slide-in navigation. Built with pure CSS using the checkbox hack.

How it works

Uses a hidden checkbox input + CSS sibling selectors (~) to toggle the menu state. No JavaScript required. The hamburger animates to an X when checked.
